## 🌐 About the Project

**අපි යමු.lk** is an interactive web application designed to help users explore, search, and stay updated with events happening across Sri Lanka — from concerts and festivals to workshops and sports events.

**Theme Selected:** Local Event Guide

---

## 🚀 Features Implemented

| Feature | Description |
|---|---|
| 🔍 Dynamic Content Updates | Live event filtering by category + real-time search |
| 🖼️ Interactive Image Slider | Auto-playing carousel with manual controls (Bootstrap) |
| ✅ Form Validation | RSVP & Contact forms with real-time error messages |
| 📜 Smooth Scrolling | CSS `scroll-behavior: smooth` + JS for anchor links |
| 🎯 Event Handling | Hover effects, modal popups, click events, navbar scroll |
| ✨ Custom Animations | Scroll-triggered fade-in-up, counter animation, staggered cards |

## 🎨 Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, animations, flexbox/grid
- **Bootstrap 5.3** — Responsive grid, navbar, carousel, modal
- **JavaScript (ES6+)** — IntersectionObserver, DOM manipulation, form validation

---

## 🖥️ Pages

1. **Home** (`index.html`) — Hero section with auto-play slider, upcoming events, testimonial, why-us section
2. **Events** (`events.html`) — Searchable, filterable grid of 9+ events with RSVP modal
3. **Contact Us** (`contact.html`) — Validated contact form with info cards and social links
